# Deep Space Troubl.io

*A pixelated .io multidirectional shooter with MOBA elements where 2 factions battle to establish control over mining operations in deep space.*


## Core Features

* Real Multiplayer!
* Epic Pixelated Aesthetics With Glow!
* Short Play Sessions (5 to 10 mins)!
* Ship Upgrades!
* Asteroids!
* Tractor Beams!
* Amazing Weapons - Lasers, Missiles and R A I L G U N
* Cool Backgrounds!
